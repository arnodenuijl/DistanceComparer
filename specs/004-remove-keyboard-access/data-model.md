## Data Model Not Applicable

This feature involves **removing** functionality rather than adding data entities. No data modeling is required.

### What This Feature Removes

This feature does not introduce new data models, entities, or state. It removes:
- UI state (focus tracking in useMapKeyboard)
- Event handlers (keyboard listeners)
- Configuration constants (KEYBOARD_PAN_STEP, KEYBOARD_ZOOM_STEP)
- Accessibility annotations (ARIA attributes in DOM)

### No New Entities

**N/A** - Removal features do not require data modeling.
